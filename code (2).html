Projetos

Confira alguns dos meus projetos no ArtStation. Aqui estão destaques selecionados:

Projeto 01 - War Radio

Mais um asset game ready focado em jogos de guerra. Esse é outra tarefa do curso de modelagem 3D da @mentorama nas aulas do @lucas.silva que me ajudou muito com feedbacks durante o processo (mesmo sendo além do que era pedido). Modelo feito em Blender (acompanhando as aulas em Maya), com menos de 30k de triângulos, texturizado e bakeado no Substance Painter (contando com texturas em 2 e 4k) em malhas distintas.

Projeto 02 – Dermatric

Alguns renders de uma nova linha de produtos da Dermatric, que tive o prazer de desenvolver em colaboração com os membros do Cubo Estúdio. Foi uma ótima oportunidade de unir design e realismo para valorizar cada detalhe da marca.

Projeto 03 – Western Station

Este foi um projeto de modelagem para uma arte mais estilizada, focando em entender melhor como passar uma "perspectiva forçada" para o 3D. Para esse propósito usei de referencia o concept do @ZC WANG. Este modelo game ready foi feito inteiramente em Blender e conta com 13,5k de triângulos (algo que fiquei muito fez de conseguir alcançar). Modelo também possuí todas os UV maps abertos para possíveis estudos de texturização futuros. Link para o concept.

Projeto 04 - Super Minigames Party!

Aqui está mais um projeto incrível para a comunidade de Fortnite no qual tive a honra de participar! Trabalhei com a talentosa equipe da "Once Creative" e o criador de conteúdo "Derponce" para trazer à vida este mapa cheio de mini games divertidos. Esses são apenas alguns dos modelos que criei, e espero que curtam! Se quiserem ver mais de perto, entrem e joguem! Código do mapa: 4960-6953-8825
